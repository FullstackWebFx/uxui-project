Інструменти, які ви опануєте

Практичні навички з інструментів індустрії, які використовуються у реальних проєктах. Кожна картка — короткий опис, а всередині модалки — детальні приклади використання, вправи та навчальні ресурси.

Приклади використання — технології

Тут зібрані навчальні приклади, поетапні вправи та корисні ресурси для кожного інструменту. Заглушки фото — замінюй на свої зображення у директорії /images/.

Figma — швидкий воркфлоу: від вайрфрейму до прототипу

1. Створення структури сторінки → 2. Атоми/компоненти → 3. Прототипування та інтерактивність → 4. Передача в розробку.

Miro / FigJam — воркшоп та карта користувача

Приклад сесії: збір потреб користувачів → картування шляхів → постановка гіпотез для A/B тестів.

Notion — організація проектної документації

Створюємо шаблон для продуктового беклогу, дизайн-система page, та template для handoffs.

HTML / CSS / JS — від дизайну до реалізації

Практика: адаптивна сітка → компонентні стилі → прості анімації → підключення простого бекенду (fetch).

Ресурси та приклади
